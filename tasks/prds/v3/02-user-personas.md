# 02. User Personas & Stories

> **Parent**: [Master PRD](./00-master-prd.md) | **Version**: 3.0.0

## 2.1 User Personas

### Mentor (콘텐츠 생성자)

| 속성 | 내용 |
|------|------|
| **역할** | 현업 담당자, 팀 선배 |
| **Pain Points** | 교육 자료 만들 시간 없음, 문서 포맷팅 스트레스 |
| **Needs** | 말/텍스트만 던지면 알아서 정리 |
| **Tech Level** | 중간 (일반 오피스 도구 사용) |

**Persona Quote**:
> "나도 바쁜데 신입 교육 자료까지 만들라고? 그냥 내가 아는 거 대충 적어주면 AI가 정리해줬으면..."

---

### Mentee (학습자)

| 속성 | 내용 |
|------|------|
| **역할** | 신입사원, 인턴, 부서 이동자 |
| **Pain Points** | 인수인계 자료 중구난방, 누구에게 물어볼지 모름 |
| **Needs** | 체계적인 커리큘럼, 핵심 요약, 이해도 확인 |
| **Tech Level** | 높음 (MZ세대, 앱 사용 익숙) |

**Persona Quote**:
> "위키가 10개나 있는데 어디서부터 봐야 하지? 내가 진짜 이해한 건지 확인하고 싶어."

---

### Admin (관리자)

| 속성 | 내용 |
|------|------|
| **역할** | HR 담당자, 팀장, 교육 담당자 |
| **Pain Points** | 신입 적응 상태 파악 어려움, 교육 품질 편차 |
| **Needs** | 정량적 데이터, 대시보드, 리포트 |
| **Tech Level** | 중간 |

**Persona Quote**:
> "OJT 잘 되고 있는지 팀장한테 물어봐도 '잘 되고 있어요'만 해서... 수치로 보고 싶어."

## 2.2 User Stories

### Mentor Stories

| ID | Story | Acceptance Criteria |
|----|-------|---------------------|
| US-M01 | 메모나 녹음 내용을 업로드하여 매뉴얼을 만들고 싶다 | 업로드 후 1분 내 초안 생성 |
| US-M02 | 생성된 자료에서 틀린 부분만 수정하고 배포하고 싶다 | 위키형 에디터, AI 수정 제안 |
| US-M03 | 내가 만든 자료가 얼마나 활용되는지 보고 싶다 | 조회수, 학습 완료율 표시 |
| US-M04 | URL만 입력하면 해당 웹페이지 내용으로 자료를 만들고 싶다 | CORS 프록시 + AI 요약 |

### Mentee Stories

| ID | Story | Acceptance Criteria |
|----|-------|---------------------|
| US-L01 | 교육 자료 학습 후 이해도 확인 시험을 보고 싶다 | 학습 완료 시 5~10문항 퀴즈 |
| US-L02 | 틀린 문제의 해설과 관련 내용을 보고 싶다 | 오답 노트 + 본문 링크 |
| US-L03 | 내 학습 진도를 한눈에 보고 싶다 | 진도율 그래프, 완료 문서 목록 |
| US-L04 | 오프라인에서도 학습을 이어가고 싶다 | Dexie.js 로컬 캐시 |

### Admin Stories

| ID | Story | Acceptance Criteria |
|----|-------|---------------------|
| US-A01 | 부서별 OJT 진행 상황을 한눈에 보고 싶다 | 팀별 진척도 그래프 |
| US-A02 | 누가 교육을 잘 시키는지 알고 싶다 | 멘토별 피드백/학습완료율 |
| US-A03 | 사용자 역할을 관리하고 싶다 | 역할 변경, 승인/거절 |
| US-A04 | 시스템 사용 현황 리포트를 받고 싶다 | 주간/월간 리포트 |

## 2.3 User Journey

### Mentor Journey

```
로그인 → 역할 선택 → MentorDashboard
    │
    ├─▶ [새 자료 만들기]
    │       │
    │       ├─ 직접 작성
    │       ├─ URL 입력 → AI 추출
    │       └─ PDF 업로드 → AI 분석
    │               │
    │               ▼
    │       AI 자료 생성 (섹션 + 퀴즈)
    │               │
    │               ▼
    │       편집 → 저장 → 배포
    │
    └─▶ [내 자료 관리]
            │
            └─ 수정 / 삭제 / 통계 확인
```

### Mentee Journey

```
로그인 → 역할 선택 → MenteeList (로드맵)
    │
    └─▶ [문서 선택] → MenteeStudy
            │
            ├─ 섹션별 학습
            │       │
            │       ▼
            │   학습 완료 버튼
            │       │
            │       ▼
            │   퀴즈 응시 (4문항)
            │       │
            │       ▼
            │   결과 확인 (3/4 통과)
            │       │
            │       └─ 오답 해설
            │
            └─ 다음 문서로 이동
```
